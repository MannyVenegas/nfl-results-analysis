# nfl-results-analysis
This is an analysis of the last 11 NFL Regular Season results. We go through the favorite, underdog and spread wins, to see if we can identify trends or patterns, that can help us make a decision when picking one.
In the .ipbyn file you can check the cleaning process, it was done in Python.
In the .PDF file you can find our findinds and the conclusions we drew up.
The Data Viz was made in Tableau. you can check it https://public.tableau.com/views/NFL2013-2023SPREADSANALYSIS/Hoja36?:language=es-ES&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link
The raw data was obtained here, go check it out https://www.kaggle.com/datasets/tobycrabtree/nfl-scores-and-betting-data/data
